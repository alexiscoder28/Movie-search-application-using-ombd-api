# Movie Search App

Simple movie search using OMDB API and React.

## Setup

1. Clone the repo
2. `npm install`
3. Create `.env` file with `VITE_OMDB_API_KEY=your_api_key`
4. `npm run dev`

## What it does

- Search movies
- Shows posters and stuff
- Debounced search so it doesn't spam the API

Built with React + Vite + Tailwind.
